# Demo: Poor Conrad's GitOps

### 1. Build The Agent

```
docker build -t my-agent:v1 agent
```

### 2. Install The Agent

```
kubectl -n default apply -f agent/install.yaml
```

Wait for the agent to apply your manifests:

```
watch kubectl -n default get all
```
